# Weather-App
This is a simple weather website that shows the current temperature and weather condition for any city you enter. It works fully on the browser and does not need any backend.

output link:-  https://viranjali1690.github.io/Weather-App/

Technologies:- 

HTML – page structure

CSS – basic styling (custom + Bootstrap)

Bootstrap 5 – responsive design

JavaScript – logic and API calls

Open-Meteo API –

     Geocoding API (city → latitude & longitude)
     
     Weather API (current weather data)
     
     No API key is required...

Project Files:-

   index.html → main webpage
   
   style.css → styling (you can customize it)
   
   script.js → JavaScript logic + API calls

How this weather app works:-

  1.User enters a city name
  
  2.App gets latitude & longitude using Open-Meteo Geocoding API

  3.Using those coordinates, it fetches current weather data
  
  4.Temperature and weather condition (with emoji) are shown on the screen


How to run / open this website:-

Method 1: Simple way

    1.Download or copy all files
    
    2.Keep index.html, style.css, and script.js in the same folder
    
    3.Double-click on index.html
    
    4.The website will open in your browser


Method 2: Using Live Server (optional)

   1.Open the folder in VS Code
   
   2.Install Live Server extension
   
   3.Right-click index.html → Open with Live Server


Features:-

  1.Search weather by city name
  
  2.Shows temperature in °C
  
  3.Weather description with emojis
  
  4.Mobile-friendly design
  
  5.No login, no API key, no backend


Requirements:-

   Internet connection is required because Weather data comes from open-meteos public API

This project is best for learning JavaScript + API integration
