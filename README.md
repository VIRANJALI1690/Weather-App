# Weather-App
This is a simple weather website that shows the current temperature and weather condition for any city you enter. It works fully on the browser and does not need any backend.

Technologies:- 

HTML – page structure

CSS – basic styling (custom + Bootstrap)

Bootstrap 5 – responsive design

JavaScript – logic and API calls

Open-Meteo API –

Geocoding API (city → latitude & longitude)

Weather API (current weather data)

No API key is required...



Project Files:-

index.html → main webpage

style.css → styling (you can customize it)

script.js → JavaScript logic + API calls



How this weather app works:-

User enters a city name

App gets latitude & longitude using Open-Meteo Geocoding API

Using those coordinates, it fetches current weather data

Temperature and weather condition (with emoji) are shown on the screen



How to run / open this website:-

Method 1: Simple way

Download or copy all files

Keep index.html, style.css, and script.js in the same folder

Double-click on index.html

The website will open in your browser


Method 2: Using Live Server (optional)

Open the folder in VS Code

Install Live Server extension

Right-click index.html → Open with Live Server



Features:-

Search weather by city name

Shows temperature in °C

Weather description with emojis

Mobile-friendly design

No login, no API key, no backend


Notes:-

Internet connection is required

Weather data comes from open-meteos public API

This project is best for learning JavaScript + API integration
